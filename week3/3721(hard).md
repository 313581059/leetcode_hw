* 解題思路 \
  初始化了三個大小為 4 倍 $N$ 的陣列：tree_min、tree_max 和 lazy，用來維護區間的最小值、最大值，以及延遲更新的標記。\
  設計了第一個輔助函數 push\
  把『當前節點還沒更新下去的懶惰標記』，發送給左邊和右邊的子節點。\
  要遞迴往下進入子節點之前，都必須先呼叫 push，確保子節點拿到的數值是最新的，最後再把當前節點的標記清空。
  
  第二個函數是 update，用來把某個區間內的所有數值同時加上一個特定的值。這裡分為三個步驟來判斷：\
  不合法的範圍，直接 return 結束\
  區間，『剛剛好』等於我們要更新的目標區間，把當前節點的 tree_min、tree_max 更新加上這個值，並記錄在 lazy 陣列裡\
  區間只有部分交疊，先呼叫 push 把舊標記推下去，然後算出中點 tm。\
  接著，把左半邊的任務交給左子樹，右半邊的任務交給右子樹。等左右子樹都更新完之後，我再拿它們的結果，來更新當前節點的最小值和最大值。
  
  第三個函數 find_first_zero 的目標是：在指定的範圍內，找到最左邊、且數值剛好等於 0 的索引位置。這裡同樣有三個判斷條件：\
  條件一：超過範圍 找不到 return -1 \
  條件二：如果這個區間的最小值大於 0（代表全都是正數），或者最大值小於 0（代表全都是負數），那這個區間裡面絕對不可能有 0，也是直接回傳 -1 提早結束。\
  條件三（到達葉節點）：如果左邊界等於右邊界，代表我們走到最底層了。這時候我們只要檢查這個點的值是不是 0：如果是 0，就回傳它的索引；如果不是 0，才回傳 -1。\
  如果上面三個條件都沒觸發，代表 0 藏在下面的子樹裡。我會先呼叫 push，然後優先往左子樹找（因為我們要找最左邊的 0）。如果左邊找到了，就直接回傳；如果左邊沒找到（拿到 -1），我才會去右子樹找。」

  主程式：\
  迴圈從左到右掃描整個陣列，\
  字典(last_pos) 記錄每個數字上一次出現的位置。\
  更新與查詢：所以我呼叫 update，把這個數字的 +1 或 -1 加到合法的左端點區間上。接著，馬上呼叫 find_first_zero，在所有可能的左端點中，找出最左邊總和為 0 的那個點。\
  更新答案：如果有找到這個點，我就用『當前的右端點』減去『找到的左端點』加 1，算出長度，並隨時更新最大長度。最後回傳這個最大值就完成了。」
  
* 程式
```python
class Solution(object):
    def longestBalanced(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        n = len(nums)
        tree_min = [0] * (4 * n)
        tree_max = [0] * (4 * n)
        lazy = [0] * (4 * n)
        
        def push(v):
            if lazy[v] != 0:
                lazy[2*v] += lazy[v]
                tree_min[2*v] += lazy[v]
                tree_max[2*v] += lazy[v]
                lazy[2*v+1] += lazy[v]
                tree_min[2*v+1] += lazy[v]
                tree_max[2*v+1] += lazy[v]
                lazy[v] = 0
                
        def update(v, tl, tr, l, r, add):
            if l > r:
                return
            if l == tl and r == tr:
                tree_min[v] += add
                tree_max[v] += add
                lazy[v] += add
            else:
                push(v)
                tm = (tl + tr) // 2
                update(2*v, tl, tm, l, min(r, tm), add)
                update(2*v+1, tm+1, tr, max(l, tm+1), r, add)
                tree_min[v] = min(tree_min[2*v], tree_min[2*v+1])
                tree_max[v] = max(tree_max[2*v], tree_max[2*v+1])
                
        def find_first_zero(v, tl, tr, limit):
            if tl > limit:
                return -1
            if tree_min[v] > 0 or tree_max[v] < 0:
                return -1
            if tl == tr:
                return tl if tree_min[v] == 0 else -1
            push(v)
            tm = (tl + tr) // 2
            res = find_first_zero(2*v, tl, tm, limit)
            if res != -1:
                return res
            return find_first_zero(2*v+1, tm+1, tr, limit)

        last_pos = {}
        max_len = 0
        
        for i, x in enumerate(nums):
            val = 1 if x % 2 == 0 else -1
            prev = last_pos.get(x, -1)
            last_pos[x] = i
            update(1, 0, n-1, prev + 1, i, val)
            left_idx = find_first_zero(1, 0, n-1, i)
            if left_idx != -1:
                max_len = max(max_len, i - left_idx + 1)
                
        return max_len
