### 3165. Maximum Sum of Subsequence With Non-adjacent Elements

* 解題思路
  設計三個函數 
  1. Build \
     初始化一棵segment tree \
     從葉子節點遞迴，向上建樹， \
     葉子存值，父節點存該區間可能的值， \
     左、右 x 選、不選 = 2x2 = 4種可能狀態
  2. Merge \
     目的是合併左子樹跟右子樹，\
     原本相鄰的元素，不能同時被選，這種case會被跳過，\
     此外的case會被記錄存回一個長度為4的array
  3. update \
     從root開始往下遞迴，修改需要換值的子樹和葉節點，\
     往上要套用merge做更新\
主程式\
build(根id,0,n-1) \
for迴圈做queries的update 回傳最大值當ans
* 程式
```python
class Solution(object):
    def maximumSumSubsequence(self, nums, queries):
        """
        :type nums: List[int]
        :type queries: List[List[int]]
        :rtype: int
        """

        n = len(nums)
        tree = [[0,0,0,0] for _ in range(4*n)]
        
        def build(idx, l, r):
            if l == r:
                v = max(0, nums[l])
                tree[idx] = [0, 0, 0, v]
                return
            
            mid = (l+r)//2
            build(idx*2+1, l, mid)
            build(idx*2+2, mid+1, r)
            tree[idx] = merge(tree[idx*2+1], tree[idx*2+2])
        
        def merge(L, R):
            res = [0]*4
            
            res[0] = max(L[0]+R[0], L[1]+R[0], L[0]+R[2])
            res[1] = max(L[0]+R[1], L[1]+R[1], L[0]+R[3])
            res[2] = max(L[2]+R[0], L[3]+R[0], L[2]+R[2])
            res[3] = max(L[2]+R[1], L[3]+R[1], L[2]+R[3])
            
            return res
        
        def update(idx, l, r, pos, val):
            if l == r:
                tree[idx] = [0,0,0,max(0,val)]
                return
            
            mid = (l+r)//2
            if pos <= mid:
                update(idx*2+1, l, mid, pos, val)
            else:
                update(idx*2+2, mid+1, r, pos, val)
            
            tree[idx] = merge(tree[idx*2+1], tree[idx*2+2])
        
        build(0,0,n-1)
        
        MOD = 10**9 + 7
        ans = 0
        
        for pos, val in queries:
            update(0,0,n-1,pos,val)
            ans = (ans + max(tree[0])) % MOD
        
        return ans
