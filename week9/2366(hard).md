### 2366. Minimum Replacements to Sort the Array

<img width="2860" height="1266" alt="image" src="https://github.com/user-attachments/assets/c8768d6a-6ddd-4112-be6a-6ec7e69cff8d" />

* 解題思路\
  初始化ans = 0；控制數字limit = 最右邊數字\
  for迴圈 從倒數第二個數字開始往回到第一個數字：\
  if nums數字沒有大於limit：\
  limit重新設定為目前nums數字\
  else：\
  決定一共要拆成多少塊不會大於控制數字的小數字，計算方式是向上取整數\
  ans += 拆幾塊-1 代表操作幾次\
  limit更新拆完之後允許最大的數字(所有拆法都安全的上限)，等一下都不可以大於它\
  return ans

* 程式
```python
class Solution(object):
    def minimumReplacement(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        ans = 0
        limit = nums[-1]

        for i in range(len(nums) - 2, -1, -1):
            if nums[i] <= limit:
                limit = nums[i]
            else:
                k = (nums[i] + limit - 1) // limit  # ceil
                ans += k - 1
                limit = nums[i] // k

        return ans
