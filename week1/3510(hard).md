* 解題思路\
根據這題的其中提示4是要注意左右鄰居，搜尋相鄰兩個數字的最小和 \
初始化：建立heap、linked list、valid list、inv \
linked list是鄰居索引 左鄰居或右鄰居 \
用python的heapq工具 取得最小的數字和，加上索引 存到pq(piority queue) list 

寫一個大的while loop 只要pq list 不是空的 \
取最小的數字和 \
檢查這個index 還是不是valid \
三個if判斷這裡有沒有涉及非遞增，有的話inv-1 (處理完這組數字和 紀錄還剩下多少inv要處理) \
把這對數字和相加起來 右邊元素索引valid改為False \
調整linklist，更新L左鄰居跟R右鄰居的元素 \
再次check這組新的有無符合遞增，抓到有問題要處理的 inv+1 \
操作一次count+=1 \
直到inv=0 return count \
數字和push到heap裡面

* 程式
```python
import heapq
class Solution(object):
    def minimumPairRemoval(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        n = len(nums)
        if n <= 1: 
            return 0
        
        # 統計初始的逆序對數量 (左邊大於右邊的相鄰組合)
        inv = sum(1 for i in range(n - 1) if nums[i] > nums[i + 1])
        if inv == 0:
            return 0
            
        # 陣列模擬雙向鏈結串列
        L = [i - 1 for i in range(n)]
        R = [i + 1 for i in range(n)]
        V = list(nums)
        valid = [True] * n
        
        # 將初始的所有相鄰數對放入 Min-Heap
        pq = []
        for i in range(n - 1):
            heapq.heappush(pq, (V[i] + V[i + 1], i))
            
        ops = 0
        while pq:
            s, i = heapq.heappop(pq)
            
            # 延遲刪除 (Lazy Deletion)：確保左元素還有效，且它當前的右鄰居還是我們要合併的對象
            if not valid[i]: continue
            j = R[i]
            # 如果 j 越界、j 已失效、或者當前的值加總已經不是 s，代表這是過期數據，直接跳過
            if j >= n or not valid[j] or V[i] + V[j] != s: continue
            
            # 準備合併 i 和 j，先取得它們外圍的鄰居
            left = L[i]
            right = R[j]
            
            # 扣除因為這次合併而即將消失的舊逆序對
            if left >= 0 and V[left] > V[i]: inv -= 1
            if V[i] > V[j]: inv -= 1
            if right < n and V[j] > V[right]: inv -= 1
            
            # 執行合併：將值累加到左元素 i，標記 j 為失效
            V[i] += V[j]
            valid[j] = False
            
            # 更新鏈結串列指標
            R[i] = right
            if right < n:
                L[right] = i
                
            # 增加因為產生新值而可能出現的新逆序對
            if left >= 0 and V[left] > V[i]: inv += 1
            if right < n and V[i] > V[right]: inv += 1
            
            ops += 1
            
            # 如果逆序對歸零，代表已經完全非遞減，提早結束
            if inv == 0:
                return ops
                
            # 將合併後產生的新相鄰數對推入 Heap
            if left >= 0:
                heapq.heappush(pq, (V[left] + V[i], left))
            if right < n:
                heapq.heappush(pq, (V[i] + V[right], i))
                
        return ops
 ```
