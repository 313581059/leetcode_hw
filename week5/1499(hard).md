### 1499. Max Value of Equation

<img width="2858" height="1275" alt="image" src="https://github.com/user-attachments/assets/cca841fa-2941-40d8-8829-19ff0eff64c4" />

* 解題思路\
    deque是可以在頭尾雙端只耗費O(n)進行快速加入/移除元素的資料結構\
    如果不用deque會變成每個j都要往前找所有i 會變成O(n ^ 2)

    Step 1 將每個點轉換成(yi-xi,xi)存入deque\
    yi-xi 負責解題目想要最大化的部分\
    xi 用來檢查是否符合距離k的限制\
    Step 2 deque的三個判斷 \
    第一個while 移除超出k距離的point > popleft\
    第二個if 更新目前最佳答案 > dq頭部最大值\
    第三個while 移除比新的point還差的舊point > pop (讓最大值永遠在最前面) \
    Step 3 return 答案
* 程式
```python
from collections import deque
class Solution(object):
    def findMaxValueOfEquation(self, points, k):
        """
        :type points: List[List[int]]
        :type k: int
        :rtype: int
        """
        dq = deque()  # 存 (yi - xi, xi)
        res = float('-inf')

        for x, y in points:
            while dq and x - dq[0][1] > k:
                dq.popleft()

            if dq:
                res = max(res, dq[0][0] + x + y)

            while dq and dq[-1][0] <= y - x:
                dq.pop()

            dq.append((y - x, x))

        return res
