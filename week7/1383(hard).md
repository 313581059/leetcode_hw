### 1383. Maximum Performance of a Team

* 解題思路\
我先將所有工程師依 efficiency 由大到小排序，這樣在每一輪中，當前的 efficiency 就可以視為 team 的最小 efficiency。\
接著用一個 min heap 來存 speed，並維護目前選到的 speed 總和 speed sum。\
在還沒有超過 k 個人之前，我會持續把 speed 加入 heap 並累加 speed sum。\
每一輪都會計算 speed sum * 當前 efficiency，更新最大 performance。\
當人數超過 k 時，我會從 heap 中移除 speed 最小的那一個，確保留下的是 speed 最大的 k 個人，並更新 speed sum。\
再重新計算 performance，看是否需要更新答案。

* 程式碼
```python
import heapq
class Solution(object):
    def maxPerformance(self, n, speed, efficiency, k):
        """
        :type n: int
        :type speed: List[int]
        :type efficiency: List[int]
        :type k: int
        :rtype: int
        """
        MOD = 10**9 + 7
        
        # (efficiency, speed)
        people = sorted(zip(efficiency, speed), reverse=True)
        
        heap = []
        speed_sum = 0
        ans = 0
        
        for e, s in people:
            heapq.heappush(heap, s)
            speed_sum += s
            
            if len(heap) > k:
                speed_sum -= heapq.heappop(heap)
            
            ans = max(ans, speed_sum * e)
        
        return ans % MOD
